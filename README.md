## What it does

- Open any SQLite file from your device storage
- Browse tables and filter rows by column using LIKE search
- Set a row limit to keep results manageable
- Export results as CSV, TXT, or JSON
- Keeps a list of recently opened files for quick access

## What it doesn't do

This is intentionally a simple read-only viewer. It does not support writing to databases, running arbitrary SQL queries, or editing data.

## Installation

Download the latest APK from the [Releases](https://github.com/martinezanthony/android-sqlite-viewer/releases/latest) page and install it on your device.
